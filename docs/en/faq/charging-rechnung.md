# Download Supercharging Invoices
Once a day, TeslaLogger downloads the supercharging invoices that are not yet available locally.

With Windows File Explorer, these can be found at the following location:
\\raspberry\teslalogger\tesla_invoices

If, as described elsewhere, TeslaLogger is operated under a different name instead of "raspberry", use the changed name accordingly.

Please do not delete the (already copied) files here, as they will otherwise be downloaded again the next night.
