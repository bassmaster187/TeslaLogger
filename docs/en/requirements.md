# Requirements

Teslalogger is designed to run on a Raspberry Pi 3+/4. For this purpose, there is also an [image](https://teslalogger.de:8808/produkt/teslalogger-raspberry-pi4-download-image/) available to simplify installation. Additionally, it is possible to run the image in a Docker context (typically on a home NAS, eliminating the need for a Raspberry Pi). Since the project is managed as open source on GitHub, it is technically possible to run it on other platforms as well. See the appendix for some hints on this.

We describe here the standard installation on a Raspberry Pi 4. The Raspberry Pi consumes very little power, is small, and can be placed almost invisibly anywhere at home where an internet connection is available. In addition to the Raspberry Pi board itself, you need a suitable power supply, a MicroSD card for the image and data, a network cable for connecting to the home network (router), and ideally a case.

On Amazon.de there is a corresponding kit, but in principle any Raspberry Pi 4 with the appropriate power supply and a MicroSD card of at least 16 GB will do. As of 2021, the much more powerful Raspberry Pi 4 is recommended, but it is not technically required.

In the Teslalogger shop, of course, there is a ready-made kit including a pre-flashed MicroSD card: [Link](https://teslalogger.de:8808/produkt/teslalogger/)
