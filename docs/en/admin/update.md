# Update

How updates for TeslaLogger should be installed automatically can be configured in "Settings" under "Automatic Updates". Usually, once a day, a search for an update is performed if the setting is not "None". To manually trigger the search, for example after changing from "Stable" to "All", the menu item "Update" can be used. The TeslaLogger will then restart, and a corresponding message will be displayed. The update usually takes less than 5 minutes, but a backup is made before the update, which takes additional time.
