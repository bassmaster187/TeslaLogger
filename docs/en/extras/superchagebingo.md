# SuperChargeBingo

SuperChargeBingo is a game about who has more check-ins at Tesla Superchargers.
So that you don't have to check in manually, TeslaLogger does it for you.

First, create an account at [https://supercharge.bingo/#/register?pk_campaign=integration&pk_kwd=teslalogger](https://supercharge.bingo)
With the "Generate Key" button, you get an API key.

Enter this key and the username in TeslaLogger:

![IMAGE](/img/extras-scb-01.png)

On the next charging session at a Supercharger, the check-in will be executed automatically.
