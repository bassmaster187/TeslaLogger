# Restore

Restoring a backup must only be used when existing data from a previous backup needs to be imported into a new or repaired installation. All existing data will be deleted. An old backup file is specified, after which the data is restored. This can take a few minutes and must not be interrupted!
The details are described in the (Initial Setup), "TeslaLogger Setup with Existing Backup".
