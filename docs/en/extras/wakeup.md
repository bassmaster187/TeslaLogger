# Wakeup TeslaLogger

This function is only needed when:
- TeslaLogger is communicating with the car so that the car cannot go to sleep
- TeslaLogger has been put into "Suspend" state.
If TeslaLogger was put to sleep via the "Suspend TeslaLogger" function, it can be woken up with this function or with the "Start TeslaLogger" button.
